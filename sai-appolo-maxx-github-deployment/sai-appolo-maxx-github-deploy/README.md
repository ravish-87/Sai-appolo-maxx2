Welcome to the Sai Appolo Maxx website - A professional e-commerce platform for motorcycle spare parts including clutch plates, engine oils, brake systems, and more.

## Features

- Responsive design for all devices
- Product showcase with detailed descriptions
- Certificate display with preview and download functionality
- Contact form with multiple communication channels
- SEO optimized with sitemap
- Google verification

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Deployment

This project is configured for GitHub Pages deployment:

1. The project will automatically deploy to GitHub Pages when you push to the `main` branch
2. The site will be available at `https://<your-username>.github.io/<repository-name>`

## GitHub Pages Deployment

This project is set up with GitHub Actions for automatic deployment to GitHub Pages:

1. Push your code to the `main` branch

3. You can find the deployment status in the "Actions" tab
